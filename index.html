Calculator

Small Calculator Sample

Install

Via Git

$ git clone https://github.com/solid-decebal/calculator.git
$ composer install

Usage

$calculator = new App\Controller\CalculatorController();
echo $calculator->compute('3+6*2-1');

Testing

$ phpunit
$ phpspec
